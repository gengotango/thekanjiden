# The Kanji Den

Public browser build of The Kanji Den for GitHub Pages.

## How saving works

- Your progress is saved to this browser automatically.
- That browser save can be lost if site data is cleared or if you switch browsers/devices.
- Use `Load Deck` to import a saved `.json` deck.
- Use `Export Deck` to download a backup of your current progress.

Recommended habit:

1. Study normally.
2. Export a fresh JSON backup periodically.
3. Import that JSON later if you change browsers, devices, or accidentally lose browser storage.

## Included features

- Browser autosave/local memory
- JSON import/export
- Bulk TXT import
- Bulk TXT export for kanji/word data sharing
- Offline lookup-assisted kanji authoring
- Review mode and cram mode

## Publishing

This folder is intended to be published as a static site on GitHub Pages.

## Local development

Open `index.html` directly in a browser, or serve the folder with any simple static file server.
